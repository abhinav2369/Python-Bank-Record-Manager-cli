## Overview 🚀

- A simple Bank Record manager CLI based program, it take the use of **pickle** module.
- It works on **Binary File** for storing and retrieving data.
- It can perform following operations:
    1. Create
    2. Display
    3. Search
    4. Modify
    5. Delete
    6. Append
    7. Exit
